Project Objective:

This project analyzes customer churn in the e-commerce domain using MySQL. By examining historical transactional data, customer attributes, and order behavior, it aims to identify patterns and drivers of churn. The insights help businesses implement targeted retention strategies and improve customer satisfaction.

*Dataset & Scope:

Customer transactional and behavioral data including:
- Tenure, order count, preferred payment mode, preferred order category
- Hours spent on app, distance from warehouse, satisfaction scores
- Complaints and churn status
- Additional table: customer_returns with return and refund details

*Key Project Steps:
1) Data Cleaning & Transformation
- Handle missing values and outliers
- Standardize column names and categorical values
- Create new columns: ChurnStatus and ComplaintReceived
2) Exploratory Data Analysis
- Count and compare churned vs active customers
- Analyze customer behavior based on city tier, payment mode, and order category
- Segment customers by warehouse distance, marital status, and coupon usage
3) Advanced Queries & Insights
- Identify churn drivers based on complaints, tenure, and purchase behavior
- Determine top preferred payment modes and order categories
- Calculate total cashback, order hikes, and app usage statistics
- Query customer_returns for churned customers with complaints

*Key Insights Generated:
- Distribution of churned vs active customers
- Relationship between complaints and churn
- Preferred payment modes and order categories of different customer segments
- City tier and distance-based churn analysis
- Behavior of high-value or frequent customers

*Conclusion:

The E-Commerce Customer Churn Analysis project demonstrates how MySQL and structured queries can be used to understand customer behavior and identify churn patterns. By analyzing transactional data, complaints, payment preferences, and order trends, the project provides actionable insights to improve customer retention, optimize service strategies, and enhance long-term business profitability.
